## How to View Tables

1. Go to the **Home** page.
2. Click the **Input Data** button, which can be found below the page title or in the top-right corner of the window.
3. Select the type of data (like Events, Students, Faculty) you wish to view. A search bar is also integrated into the page for your convenience.
4. You will be directed to the page displaying the records of data associated with the selected type in tabular format.
5. You can select the **Input Data** button below the table heading to create a new record in the table.

## Table Schema

<TypeTable
  type={{
    id: {
      type: 'uuid',
      description: 'Primary key of the table, uniquely identifies a user.',
    },
    first_name: {
      type: 'varchar(100)',
      description: 'The first name of the user.',
    },
    middle_name: {
      type: 'varchar(100)',
      description: 'The middle name of the user (if any).',
    },
    last_name: {
      type: 'varchar(100)',
      description: 'The last name of the user.',
    },
    email: {
      type: 'varchar(255)',
      description: 'The email address of the user.',
    },
  }}
/>

## Relationships

- The `id` field in this table is referenced by:
  - `faculty_id` in the `faculty_courses` table.
  - `awardee_id` in the `ipr` table.

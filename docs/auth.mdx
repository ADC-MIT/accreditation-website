title: Auth
description: This page documents the `auth` table in the database schema. The `auth` table stores authentication details such as passwords and last login timestamps for users.
---


## Table Schema

<TypeTable
  type={{
    id: {
      type: 'uuid',
      description: 'Primary key of the table, references the users table.',
    },
    password: {
      type: 'varchar(255)',
      description: 'Hashed password for authentication.',
    },
    last_login: {
      type: 'timestamp',
      description: "Timestamp of the user's last login. Can be null.",
    },
    requires_reset: {
      type: 'boolean',
      description:
        'Indicates whether the user must reset their password upon login.',
      default: 'false',
    },
  }}
/>

## Relationships

- The `id` field references the `id` field in the `users` table.

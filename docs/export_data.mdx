## How to Export Data

1. Go to the **Home** page.
2. Click the **Export Data** button, which can be found below the page title or in the top-right corner of the window.
3. Select the ranking body (NAAC, NBA, NIRF, etc.) whose format you wish to export the data in.
4. A dropdown menu appears, displaying various categories of data specific to the selected ranking body. Select your desired category of data to be exported.
5. You will be directed to a page displaying the records of data for the selected category.
6. Click on the **Export Data** button on the top right corner, just below the category heading. A copy of the data will be downloaded to your local computer in **PDF** format.

### Exporting Data Manually

1. For ranking bodies other than the ones listed, select the **Export Data Manually** option.
2. Select the type of data you wish to be exported (Events, Faculty, Students).
3. You will be directed to the page displaying the records for the selected type.
4. Click on the **Export Data** button to get a copy of the data in **PDF** format.

## Example Workflow for Exporting Data

1. To get started, click on the **Export Data** button and select the **NAAC** format.
2. In the dropdown menu, select **NAAC (3.4.3)** to export patent and copyright data.
3. You will be directed to the page that contains records of patents and copyrights issued to the institution and its members.
4. Click on the **Export Data** button to get a copy of the data in **PDF** format.

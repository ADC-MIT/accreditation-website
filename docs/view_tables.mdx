## How to input data ?

- Go to the **Home** page.
- Click the **Input Data** button which can be found below the page title or in the top-right corner of the window.
- Next, select the type of data (like events, students, faculty) you wish to view. A search bar is also integrated into the page for your convenience.
- You will now be directed to the page displaying the records of data associated with the selected type in tabular format.
- You can select the **Input data** button below the table heading to create a new record in the table.

## Table Schema

<TypeTable
  type={{
    student_id: {
      type: 'uuid',
      description: 'Primary key, references a student.',
      relation: 'students.id',
    },
    semester: {
      type: 'int',
      description: 'The semester number.',
    },
    GPA: {
      type: 'decimal(4,2)',
      description: 'The Grade Point Average for the semester.',
    },
    credits: {
      type: 'int',
      description: 'The total credits earned in the semester.',
    },
  }}
  indexes={[
    {
      fields: ['student_id', 'semester'],
      unique: true,
      description: 'Ensures a unique GPA entry per student per semester.',
    },
  ]}
/>

## Relationships

- The `student_id` field references the `id` field in the `students` table.

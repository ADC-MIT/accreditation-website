## Table Schema

<TypeTable
  type={{

    },
    speaker: {
      type: 'text',
      description: 'The name of the speaker at the event.',
    },
    no_of_students: {
      type: 'int',
      description: 'The number of students who attended the event.',
    },
    no_of_teachers: {
      type: 'int',
      description: 'The number of teachers who attended the event.',
    },
    link_to_report: {
      type: 'text',
      description: 'A link to the event report.',
    },
    geotagged_photographs: {
      type: 'text[]',
      description: 'An array of links to geotagged photographs from the event.',
    },
  }}
/>

## Relationships

- The `id` field references the `id` field in the `events` table.
